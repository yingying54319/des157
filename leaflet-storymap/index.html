Leaflet GeoJSON Selector

Leaflet Control for select GeoJSON properties in a interactive menu and map

Other useful stuff for Web Mapping...

Features

  • List in menu GeoJSON properties
  • Customize item text by properties
  • Locate geometry on item custom events
  • Reolad GeoJSON at runtime
